# Scientific Writing Skill

Structured guidance for drafting and revising scientific manuscripts, abstracts, cover letters, and reviewer responses. Use this skill when writing or editing scholarly work to keep clarity, statistical rigor, and reproducibility front and center.

3. Use the references for section patterns, revision passes, and response templates.
